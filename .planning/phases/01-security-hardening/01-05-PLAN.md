<objective>
Implement the server-issued problem token (SEC-10). The server "deals" problems — issue_problem creates an IssuedProblem row and surfaces the token via a private IssuedProblemResult table. submit_answer verifies the token before accepting the answer. SprintPage is updated to call issue_problem first and pass the token.

Purpose: Closes the final score injection vector — an attacker can no longer call submit_answer with arbitrary problem pairs because the server must have previously issued that exact (session_id, a, b) combination.
Output: Full server + client integration of the token flow. This is the most invasive plan in Phase 1.

IMPORTANT — Open question from STATE.md / RESEARCH.md:
The client uses subscribeToAllTables(). Private tables are NOT included in subscribeToAllTables(). Before implementing, verify whether `conn.subscriptionBuilder().subscribe(['SELECT * FROM issued_problem_results WHERE owner = X'])` works for a private table in SpacetimeDB 2.x. If targeted subscription to a private table does not work, the fallback is to make IssuedProblemResult a public table with row-level filtering (the client ignores rows that don't belong to it — acceptable since the table only contains short-lived tokens, not sensitive secrets). Document the finding in the SUMMARY.
</objective>

    **Step 4: Add issue_problem reducer:**
    ```rust
    #[reducer]
    pub fn issue_problem(ctx: &ReducerContext, session_id: u64, a: u8, b: u8)
        -> Result<(), String>
    {
        let session = ctx.db.sessions().id().find(session_id)
            .ok_or("Session not found")?;
        if session.player_identity != ctx.sender() {
            return Err("Not your session".into());
        }
        if session.is_complete {
            return Err("Session already complete".into());
        }
        let pair_tier = pair_learning_tier(a, b).ok_or("Invalid problem pair")?;
        let player = get_player(ctx)?;
        if pair_tier > player.learning_tier {
            return Err("Problem pair above player's current tier".into());
        }
        let token = make_code(ctx);
        ctx.db.issued_problems().insert(IssuedProblem {
            id: 0, session_id, a, b, token: token.clone(),
        });
        // Write token to result table for client to read
        match ctx.db.issued_problem_results().owner().find(ctx.sender()) {
            Some(_) => {
                ctx.db.issued_problem_results().owner().update(IssuedProblemResult {
                    owner: ctx.sender(),
                    token,
                });
            }
            None => {
                ctx.db.issued_problem_results().insert(IssuedProblemResult {
                    owner: ctx.sender(),
                    token,
                });
            }
        }
        Ok(())
    }
    ```
